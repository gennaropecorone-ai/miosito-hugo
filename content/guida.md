# Guida Completa: G&G Cyber-Lab

Questa pagina contiene il riepilogo dei passaggi per la gestione del blog.

## 1. Configurazione Iniziale
- **Tema:** Hugo Coder
- **Stile:** Cybersecurity (Verde Neon su Nero)
- **Cartella di pubblicazione:** `docs`

## 2. Comandi di Aggiornamento (Il Rituale)
Ogni volta che fai una modifica, esegui questi comandi nel terminale:

1. `hugo` (per generare i file aggiornati)
2. `git add .` (per preparare i file)
3. `git commit -m "Messaggio della modifica"`
4. `git push origin main` (per mandare tutto online)

## 3. Parametri Grafici nel hugo.toml
- **avatarSize:** 350px (per l'immagine grande)
- **colorScheme:** "dark"
- **baseURL:** Il link ufficiale di GitHub Pages.
